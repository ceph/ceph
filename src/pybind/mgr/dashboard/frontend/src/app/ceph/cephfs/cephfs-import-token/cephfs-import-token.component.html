Import Bootstrap Token

Paste the token generated on the remote cluster to establish a connection.

Bootstrap Token

The token contains encrypted connection details and authentication credentials for the remote cluster.

How to obtain the bootstrap token
  • Log in to the remote cluster's Ceph Dashboard.
  • Navigate to CephFS → Mirroring → Configure (Remote).
  • Complete the remote cluster setup wizard.
  • Generate and copy the bootstrap token.
  • Paste the token in the text area above.
